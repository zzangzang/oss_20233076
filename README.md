## 1 .top 
#### > top 명령어란?
> 실시간으로 CPU 사용률을 체크해주는 유틸리티로, 작업관리자와 비슷한 성격을 가지고 있다. top 명령어를 사용하여 CPU, 메모리, 디스크, 네트워크 등의 시스템 성능 지표와 실행 중인 프로세스 목록을 볼 수 있다.
